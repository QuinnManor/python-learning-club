# Python Numbers / Numeric Values (int, float)

## Syntax
**int** (whole numbers or non-decimal numbers)
```python
-100
-4
1
500
```

**float** (decimal numbers)
```python
-100.29
-4.4
1.01
500.05
```

## Numeric Conversions
You can convert numbers to other numeric types
```python
int("5") # output: 5
int(5.0) # output: 5

float("4.0") # output: 4.0
float(4) # output: 4.0
```
